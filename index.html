Update v0.1.4a - SELECTION UPDATE

Card Modes!

You can now enter two card modes: Use (use cards) & Select (select cards)
With the new card mode, select, new opportunities for cards will be available. Enemy cards can now be selected if you want to attack specific ones, specific ally cards can be healed, etc.

Discarding

Discarding has now been introduced. You get 1 discard each turn (maximum of 1) and you can discard a card by going into select mode, selecting a card, and clicking discard. You will get 1 mana back after discarding. Discarding is a new mechanic that will allow clearing up your hand if full, and also removing unnecessary cards for mana that can be used for other things.

Added 0 new cards:

  • The null card, which doesn't exist.

Other Additions:

  • Added some more card art
  • Made a few more cards action cards (some were support, but shouldn't have been)
  • The changelog now supports different screen sizes. Previously it had a set size, but its size is now responsive (90% of the viewport's size)

Balance Changes:

  • None

Bug Fixes:

  • AI Mode 3 will now continue once it starts. Previously it ended immediately after the opponent's mana went under 8. Now it will go until the enemy is destroyed.
  • Clone box now works on cards with a numbered inventory id (e.g. when there are two of the same cards, one of their ids is numbered. previously, clone box would break if that were to happen)

Next Update?

For next update, more foils are planned to be added, the final 2 card arts will be added, AI will also get special card targeting, sounds, and more is planned.
