Update v0.1.2b

Added 7 new cards:

  • Reaper, which applies 'Death' effect. This effect kills affected card after 3 turns.
  • Froster, which applies 'Frozen' effect. Affected card cannot play and will slowly lose healh and attack.
  • Ninja, which has a camouflage effect when attacking. This makes cards unable to see it.
  • Transportation Bus, which swaps the first and last card.
  • Clone Box, which clones the first card.
  • Oblivion, which appears if the clone box is the first card. Very useful.
  • Drawback, which deletes all of your cards but heals you and gives you mana based on their total health. (Mana = Total/13, Health = Total/3.5)

Balance Changes:

  • Circuit Node (now Cultist) starts with higher atk, making it more useful if there are no duplicates nearby.
